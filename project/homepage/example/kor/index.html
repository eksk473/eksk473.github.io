PAT

PUBPLE AUTOMATION TOOL

PDF 파일을 자동으로 변환하는 스마트한 트랜스코딩 솔루션

  • 백터 이미지 및 TEXT, META DATA 자동 추출
  • 자동 EPUB 생성
  • PDF/EPS 파일 포맷 사용을 통한 시간/비용 절약
  • PSE를 통한 편집 연계와 PSC로 콘텐츠를 쉽게 관리
  • PSV를 통해 즉시 서비스 가능

PSE

PUBPLE SMART EDITOR

illustrator를 이용한 편집으로 EPUB을 쉽게 제작하는 에디터

  • Adobe illustrator 의 ADD-ON 프로그램으로 제공
  • WEB 기반의 에디터 별도 서비스 제공
  • 문제풀이, 멀티미디어, 외부 심화학습 자료 등을 코딩 없이 쉽게 편집 가능
  • PSC, PSV로 바로 연계하여 서비스 가능

PSV

PUBPLE SMART VIEWER

다양한 환경에서 구동되는 스마트한 뷰어 솔루션

  • 최신 WEB KIT ENGINE 최적화로 콘텐츠 구동 속도 향상
  • PC, iOS, Android 등 Device에 구애 받지 않음
  • SVG를 TEXT로 인식하여 검색과 복사가 가능한 SVG 엔진 탑재
  • 학교 환경에서 가장 많이 사용하는 판서, 메모, 스크랩 등의
    다양한 교수학습 기능 제공

PSC

PUBPLE SMART CMS

제작한 콘텐츠를 쉽게 관리하는 CLOUD 기반의 CMS

  • CLOUD 기반의 콘텐츠 관리
  • 미리 보기와 전체 보기를 통해 콘텐츠 제공 목적에 맞게 구성 가능
  • PSV 뷰어에서 제공할 기능을 설정하여 서비스 가능
